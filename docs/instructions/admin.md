## Для администраторов

### Доступ

1. Ваш Telegram ID должен быть внесен в список администраторов
2. При запуске бота автоматически определяется ваш уровень доступа

### Панель управления

**Откройте панель администратора:**
- Команда `/admin`
- Или кнопка "Админ-панель" в главном меню

### Управление заказами

**Просмотр заказов:**
- "Заказы в ожидании" - новые необработанные заказы
- "Заказы в работе" - принятые в производство
- "Готовые заказы" - выполненные заказы
- "Отклоненные заказы" - отмененные заказы
- "Все заказы" - полный список

**Работа с конкретным заказом:**
1. Выберите заказ из списка
2. Просмотрите полную информацию:
   - Данные клиента (ФИО, Telegram)
   - Фото модели
   - Выбранные материалы
   - Дата создания
   - Текущий статус

### Скачивание моделей

1. В карточке заказа нажмите "Скачать модель"
2. Бот отправит вам файл, автоматически переименованный по формату:  
   `{НомерЗаказа}_{Фамилия}_{Имя}_{НазваниеДетали}.stp` (или `.stl`, `.step`)

**Пример:** `105_Иванов_Петр_Корпус_устройства_v2.stl`

### Изменение статусов

Доступные действия с заказом:
- **"Принять в работу"** - заказ перемещается в производство
- **"Отметить готовым"** - заказ выполнен
- **"Отклонить"** - заказ отклонен с возможностью указания причины
- **"Вернуть в ожидание"** - вернуть на доработку

### Управление материалами

**Редактирование списков:**
- Команда `/manage_materials`
- Добавление новых типов пластика
- Удаление устаревших материалов
- Редактирование палитры цветов

### Уведомления клиентов

- При смене статуса заказа клиент автоматически получает уведомление
- При отклонении заказа можно отправить комментарий с причиной

### Рекомендации по работе

1. **Регулярно проверяйте** заказы "В ожидании"
2. **Скачивайте модели** сразу при принятии в работу
3. **Своевременно обновляйте статусы** для информирования клиентов
4. **Следите за актуальностью** списков материалов и цветов

### Получение помощи

- Технические проблемы: обращайтесь к разработчику
- Вопросы по работе с ботом: используйте `/help`
