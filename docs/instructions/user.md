## Для пользователей

### Как начать работу

1. **Найдите бота** в Telegram по имени `@robo548manufacturing`
2. **Запустите бота** командой `/start`
3. **Пройдите регистрацию** - при первом использовании бот запросит ваши Фамилию и Имя

### Создание заказа

1. **Начните новый заказ** - отправьте команду `/new_order` или нажмите кнопку "Создать заказ"
2. **Загрузите фото модели**:
   - Отправьте скриншот, чертеж или фото детали
   - Можно добавить текстовое описание к фото
3. **Загрузите 3D-модель или чертеж**:
   - Поддерживаются только форматы **.STL**, **.STP**, **.DXF**
   - Другие форматы не принимаются
4. **Укажите название детали** - введите понятное название для вашей модели
5. **Выберите материал**:
   - Выберите тип пластика из предложенного списка (PLA, PETG и т.д.)
   - Выберите цвет из доступных вариантов
   - Для резки выберите толщину фанеры
6. **Подтвердите заказ** - проверьте все данные и нажмите "Подтвердить"

### Просмотр своих заказов

- Используйте команду `/my_orders` чтобы увидеть список всех ваших заказов
- Для каждого заказа отображается:
  - Номер заказа
  - Дата создания
  - Текущий статус
  - Выбранные материалы

### Уведомления

- Вы будете автоматически получать уведомления при изменении статуса заказа
- Уведомления приходят в личный чат с ботом

### Важные примечания

- Файлы моделей должны быть в формате STL, STP или DXF
- Максимальный размер файла: 50 МБ (ограничение Telegram)
- Сохраняйте копии ваших файлов у себя
